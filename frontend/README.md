# Chatbot NMK Design Frontend

Frontend Next.js cho chatbot NMK Design.

## Tính năng

- Giao diện chat hiện đại với Tailwind CSS
- Tích hợp với FastAPI backend
- Hỗ trợ session ID để theo dõi cuộc hội thoại
- Loading states và error handling
- Responsive design
- Dark mode support

## Stack công nghệ

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Axios
- Lucide React Icons
